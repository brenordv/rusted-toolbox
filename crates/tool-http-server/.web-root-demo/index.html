Historical Timeline

1962

The Vision Begins

J.C.R. Licklider of MIT proposes the "Galactic Network" concept, envisioning a globally interconnected set of computers through which everyone could quickly access data and programs from any site.

1965

First Wide-Area Connection

The first wide-area network connection is made between MIT Lincoln Lab and System Development Corporation in Santa Monica via a 1200 bps phone line, proving the feasibility of wide-area networking.

1969

ARPANET is Born

The first ARPANET link is established between UCLA and Stanford Research Institute. This marks the true beginning of the Internet as we know it, with the first message sent being "LO" (attempting to type "LOGIN").

1971

Email Revolution

Ray Tomlinson creates the first email program for ARPANET, introducing the "@" symbol to designate email addresses. This becomes one of the most transformative applications of the early Internet.

1973

TCP/IP Protocol

Vint Cerf and Bob Kahn develop the Transmission Control Protocol (TCP), which later evolves into TCP/IP. This protocol becomes the foundation for Internet communication worldwide.

1983

The Internet Protocol

ARPANET officially adopts TCP/IP protocol, creating a true "network of networks." This date is often considered the official birth of the modern Internet.

1989

World Wide Web

Tim Berners-Lee at CERN proposes the World Wide Web, creating the first web browser, web server, and website. This transforms the Internet from a text-based system to a multimedia platform.

1991

Public Access

The World Wide Web becomes publicly available, and the National Science Foundation lifts restrictions on commercial use of the Internet, paving the way for widespread adoption.
